Simply adds a nofollow tag to all links within a post.

== Description ==

WP-NofollowPost adds nofollow tags to links in a post. This is done at the content-display level, 
meaning you don't have to go back and apply this plugin to old posts. All posts will 
automatically display all links with nofollow tags in them after enabling this plugin.

== Installation ==

1. Upload `wp-nofollowpost` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You're all set... all links in all posts will automatically be changed to nofollow.
